# AI Resume Parser & Matcher

##  Project Description
The AI Resume Parser & Matcher is a web-based application that analyzes resumes and compares them with job descriptions. The system extracts skills from resumes and job descriptions and calculates a match percentage to evaluate candidate suitability.

This project is developed using Python and Flask.

---

##  Features

✔ Upload Resume (PDF or Text)  
✔ Extract Skills Automatically  
✔ Compare Resume with Job Description  
✔ Skill Gap Analysis  
✔ Match Percentage Calculation  
✔ Matching Skills Display  
✔ Missing Skills Display  

---

##  How It Works

The system performs the following steps:

1. Resume is uploaded
2. Text is extracted from resume
3. Skills are identified
4. Job description skills are extracted
5. Skills are compared
6. Match percentage is calculated

### Example

Job Description Skills:
- Python
- Machine Learning
- SQL
- HTML

Resume Skills:
- C
- C++
- HTML
- CSS
- JavaScript
- Python

Matching Skills:
- Python
- HTML

Missing Skills:
- Machine Learning
- SQL

Match Percentage:
50%
